# AI_and_Data_Analysis_Course

本课程用于《人工智能与数据分析》系统介绍智能计算机系统设计的基本思想与核心技术，重点强调统计建模与决策理论范式。课程采用理论讲解与实践相结合的教学方式，引导学生理解并掌握在不确定环境中进行推理与决策的关键方法。
通过本课程学习，学生将能够构建自主智能体，使其在完全信息、部分可观测以及对抗性环境中高效地进行决策；掌握不确定条件下的推断方法，并针对不同奖励结构优化行动策略。同时，课程引入机器学习算法，指导学生完成手写数字与图像分类等典型人工智能任务。
本课程所涉及的人工智能与数据分析技术具有广泛的适用性，可应用于多种智能系统与数据驱动问题，为学生后续在人工智能、数据科学及相关应用领域的深入学习与研究奠定坚实基础。
本课程配套内容包括：（1）中英文双语课件；（2）与课件对应的实时讲授脚本；（3）人工智能与数据分析实验课程案例。
